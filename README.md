## Install

Open your terminal and type in:

SSH Method
$ git clone git@github.com:danielsoto-dev/vanilla-frontend-test-ideaware.git
$ cd vanilla-frontend-test-ideaware
$ npm i

To see the web page, use the Live Server Extension on VS Code.
